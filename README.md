# Smart Issue Reporter 

A simple web app to report and track issues.

## Features
- Add issues with name and description
- View all reported issues
- Simple and clean UI

## Tech Stack
- Python (Flask)
- HTML, CSS
- GitHub

## How to Run
1. Clone the repo
2. Install requirements
3. Run app.py

- Add login system
